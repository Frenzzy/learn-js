Домашнее задание №5

Переписать домашнее задание №3 с использованием конструкторов.

Эмулировать жизнь котят приговоренных к смерти.
  • Должна быть функция createKitten, которая принимает 2 аргумента: имя котенка и его слова которые он говорит перед смертью
  • функция должна возвращать объект, у которого будет свойство name и метод die
  • во время выполнения метода die котенок должен говорить свои последние слова
  • Должен существовать объект bag
  • у него должен быть метод put который принимает котенка
  • метод getKitten который принимает имя котенка и возвращает котенка с этим именем
  • метод sink во время которого погибают все котята (у них вызывается метод die)
Текст из домашнего задания №3

Пример:

var kitten = new Kitten('April', 'I wanna live');
kitten.name; // 'April'
kitten.die(); // logs 'I wanna live'

var stupidKitten = new Kitten('Nyasha', 'meow');

var bag = new Bag;
bag.put(kitten);
bag.put(stupidKitten);
bag.getKitten('April') === kitten; // true
bag.getKitten('notExistedKitten') === null; // true
bag.sink(); // logs 'I wanna live' 'meow'
