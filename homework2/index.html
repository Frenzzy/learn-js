Домашнее задание №2

Пишем функцию cascadeCall, которая должна уметь:

  • принимать на вход произвольное поличество функций
  • каждая из этих функций, кроме последней, должна принимать один аргумент, производить над ним любые операции и возвращать результат
  • последняя функция должна просто возвращать какой-либо результат
  • функция cascadeCall должна вернуть результат первой функции, в которую должен передаться результат второй функции, в которую должен передаться результат третьей и т. д.
  • cascadeCall(a, b, c, d) == a(b(c(d())))

Пример:

function a(n){
  return n*2;
}

function b(){
  return 5;
}

cascadeCall(a, b) == 10; // true
